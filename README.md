Here's my resume! I realized that I had many copies of the data that is on
my resume in different places (and I want to add another), so I decided to
define a single `.yaml` file as the source of truth. I plan to use that
to create copies of my resume in different formats (for fun, of course).

For now, you can see my resume compiled in Typst from the `.yaml` file
on my website, at [resume.renntg.com](https://resume.renntg.com).
